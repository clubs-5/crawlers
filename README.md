# crawlers
* 爬蟲結果存成 csv 檔

* 每個人把自己的爬蟲程式建立一個資料夾後，在該資料夾裡面開始寫爬蟲。資料夾名稱以目的為名。
  範例: 艾美獎黃金時段爬蟲資料夾
  emmy_primtimes

  範例: imdb 影集基本資料
  imdb_shows
